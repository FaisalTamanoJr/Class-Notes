| $F(t)$       | $L\{F(t)\}$          | Notes                       |
| ------------ | -------------------- | --------------------------- |
| $a$          | $\frac{a}{s}$        | $a=\text{constant}$         |
| $t$          | $\frac{1}{s^2}$      |                             |
| $t^n$        | $\frac{n!}{s^{n+1}}$ | $n=\text{positive integer}$ |
| $e^{at}$     | $\frac{1}{s-a}$      | $s>a$                       |
| $e^{-at}$    | $\frac{1}{s+a}$      | $s>a$                       |
| $\cos (kt)$  | $\frac{s}{s^2+k^2}$  |                             |
| $\sin(kt)$   | $\frac{k}{s^2+k^2}$  |                             |
| $\cosh (kt)$ | $\frac{s}{s^2-k^2}$  |                             |
| $\sinh(kt)$  | $\frac{k}{s^2-k^2}$  |                             |
